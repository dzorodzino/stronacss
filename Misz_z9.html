    Znacznik |link| może występować wielokrotnie.

  4. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasad musi być zachowana przy większej ilości dołączanych styli?

    • Można dołączyć dowolną liczbę arkuszy.
    • Kolejność ma znaczenie – późniejsze style mogą nadpisywać wcześniejsze.

  5. Co się dzieje w przypadku konfliktów? Przez konflikt rozumiemy, gdy te same style są definiowane w różnych arkuszach styli podłączone do tego samego arkusza.

    6. Jeśli te same style są w różnych arkuszach CSS, obowiązuje zasada kaskady:

    • Styl wczytany później ma pierwszeństwo.
    • Bardziej szczegółowy selektor wygrywa.
    • Styl z !important ma najwyższy priorytet.

  6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

    • Tworzy się je jako zwykłe pliki tekstowe.
    • Rozszerzenie: .css
    • można edytować np. w: Notatniku, Notepad++, Visual Studio Code

  7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

    • Podpowiedzi składni, kolorowanie kodu, sprawdzanie błędów, automatyczne formatowanie
    • Przykłady: Visual Studio Code, WebStorm, Brackets
