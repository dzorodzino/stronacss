1. Wymienić oraz opisz dwie części selektora,

    2. Selektor CSS składa się z dwóch głównych części:

    • Selektor – określa, do jakiego elementu HTML zastosować styl.
      Przykłady: p, .menu, #header
    • Blok deklaracji – znajduje się w nawiasach klamrowych { } i zawiera właściwości (cechy) oraz wartości przypisane do elementu.

  2. Znaczenie nawisów klamrowym {...},

    • Nawiasy klamrowe otwierają i zamykają blok deklaracji.
    • Wszystkie style dla danego selektora muszą znajdować się wewnątrz klamr.
    • Bez nich przeglądarka nie rozpozna prawidłowo stylów.
    p {
    color: red;
    }

  3. Opisz cechy,

    4. Cechy określają co chcemy zmienić w wyglądzie elementu. Przykłady:

    • color
    • font-size
    • background-color
    • margin
    • padding
    • border

  4. Opisz wartości

    5. Wartość to konkretne ustawienie cechy. Przykłady:

    color: red;
    font-size: 16px;
    margin: 10px;

  5. Kiedy stosujemy znak średnika (;), kiedy nie jest konieczny

    • Kiedy stosujemy: po każdej właściwości, jeśli jest ich więcej niż jedna.

      p {
      color: red;
      font-size: 16px;
      }

    • Kiedy nie jest konieczny: gdy w bloku znajduje się tylko jedna właściwość.

      p {
      color: red
      }
