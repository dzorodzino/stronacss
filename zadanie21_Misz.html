Co znaczy skrót ID? ID oznacza Identifier, czyli unikalny identyfikator elementu na stronie internetowej.
Jaki jest cel stosowania ID? ID służy do oznaczenia jednego konkretnego elementu na stronie oraz do nadawania mu stylu w CSS lub użycia w JavaScript.
Jakie jest znaczenie znaku # Znak # w CSS oznacza odwołanie do identyfikatora ID danego elementu.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie ID odbywa się poprzez znak # w CSS, np. #tabela oznacza styl dla elementu który ma zapis: id="tabela".
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia
  1. ID może wystąpić tylko raz na stronie.
  2. ID ma wyższy priorytet niż klasy i znaczniki.
  3. ID używamy tylko do jednego konkretnego elementu.
